# ZEXVRO

Web2-to-Web3 PaaS middleware that abstracts complex public blockchain rails into secure, private, enterprise-grade endpoints using Zero-Knowledge Execution (ZEX) and Verifiable Routing Orchestration (VRO).

ZEXVRO is planned as a unified Web3 PaaS with a clean Vercel/Cloudflare-like developer experience. The MVP focuses on six unique services before expanding into common platform capabilities such as databases, deployment, hosting, security, and connectors.

## Project Docs

- `context.md` contains the current product, team, service, and setup context.
- `memory.md` is the shared working memory for developers and agents.
- `assets/brand/` contains the current logo, typo logo, and brand design assets.

Before starting work, read `context.md` and `memory.md`. After meaningful changes, update `memory.md` and commit the memory update with the code.
