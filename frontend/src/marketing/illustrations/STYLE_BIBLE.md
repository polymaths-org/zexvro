# ZEXVRO Marketing Illustration Style Bible

- Canvas: `640x460` for chapter assets; `900x640` for the static substrate fallback.
- Base surface: `#050505`; raised blocks: `#0A0A0B` and `#111113`.
- Default stroke: `#FAFAFA`, `2px`, square/geometric caps where practical.
- Default border: `#27272A`, `2px`.
- Nodes: rounded rectangles with `6px` to `8px` radius, matching the ZEXVRO mark without becoming soft UI cards.
- Edges: straight or restrained cubic paths; no neon glows, no coin/token motifs.
- Accent usage: one accent family per chapter, used only on proof paths, boundaries, pulses, or status strokes.
- Motion rigging: each SVG keeps stable ids/classes for independent animation; never flatten animated groups into one path.
