# ZEXVRO Arcade (Lakebed)

This capsule is the **Morph demo app**.

- Baseline is **Web2** (`gameConfig.mode = web2`).
- Morph migrates toward **Web3** using ZEXVRO Gate + NFT (+ optional De-pin).
- After edits: `npx lakebed deploy` so the shared URL updates for everyone.

Do not put secrets in client code. Public Gate `siteKey` is OK; `secretKey` stays server/env only.
